User/Player is not given chance to select their selected opponent.
 - Opponent is selected by server without asking player.
 

Restart functionality is not available fully.
  - Server need not to be restarted.
  - Player need to restart the game(gui) on both the player's end.

  
If one player manually forcefully terminates the game other player is not notified about that status.  
  - In above case server will response player's opponent who forcefully terminates the game.
